# Projet HTML

Projet web réalisé entre collègues dans le but d’apprendre, expérimenter et améliorer nos compétences en HTML / CSS / JavaScript.

Le projet est simple, évolutif et sert de support de travail collaboratif.

---

## Appendix

### Convention de nommage

Le projet utilise la **Hungarian Notation** pour améliorer la lisibilité du code.

https://en.wikipedia.org/wiki/Hungarian_notation

Exemples :

- `strName` → string
- `nCount` → number
- `bIsActive` → boolean
- `arrUsers` → array
- `objConfig` → object

## Installation

Cloner le projet :

```bash
git clone https://github.com/misterbraindamage/azerty.git

Entrer dans le dossier :

cd azerty
Usage

Ouvrir simplement le fichier :

index.html

dans un navigateur.

Git Workflow
Ajouter des modifications
git add .
Commit
git commit -m "message de modification"
Récupérer les dernières versions
git pull origin main --rebase
Envoyer sur GitHub
git push origin main
Problèmes fréquents
Erreur push rejeté
git pull origin main --rebase
git push origin main
Conflits Git
Corriger les fichiers
Puis :
git add .
git rebase --continue
```
